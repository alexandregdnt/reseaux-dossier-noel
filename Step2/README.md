# Réseaux: Projet de Noël

Pour démarrer le serveur :
```
./ser 127.0.0.1 2000
```

Pour démarrer le client :
```
./cli 127.0.0.1 2100 127.0.0.1 2000
```

### Informations supplémentaires

Pour supprimer les fichiers compilés :
```
make clear
```

Pour supprimer les fichiers de données :
```
make cleardata
```

Pour supprimer les fichiers de logs :
```
make clearlogs
```
